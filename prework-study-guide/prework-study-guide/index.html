Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The DOCTYPE tag is a declaration that tells browsers what document type to expect.
  • HTML attributes are always defined in the beginning of the element but NEVER the closing element

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS gives content their style
  • CSS styles associated with a website are triggered by entering style inside angled brackets

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • A local environment is a workspace for web developers to work independently
  • An integrated development environment (IDE) is software used by developers to combine different features needed to work on a single interface.

JavaScript

  • JavaScript handles logic and allows user interaction
